## Template

```
# NNNN - Título

**Status:** proposta / aceita / substituída
**Data:** AAAA-MM-DD

## Contexto
O problema ou a força que motiva a decisão.

## Decisão
O que foi escolhido.

## Consequências
Prós e contras conhecidos, e o que fica em aberto.
```
